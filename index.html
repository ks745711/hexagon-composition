Hexagon Composition

This is a javascript-based generative web application. Use the controls on the right to manipulate the variables.

Parameters

  • Size: This controls the size of each individual hexagon.
  • Horizontal speed: This controls the speed of all hexagons currently traveling on path in a horizontal direction.
  • Vertical speed: This controls the speed of all hexagons currently traveling on path in a vertical direction
    Extra hexes: This controls the amount of individual hexagons on the screen.
